#

### 결과
#
#### - 알고리즘 문제에 대한 풀이
#####  -> 공대 5호관 402호에서 진행 완료.
# 
 
#### - 서로에 대한 피드백
#####  -> 서로가 풀어온 방법으로 피드백 완료.
#####  -> 다음주차는 설연휴여서 진행을 안하고 그다음주에 진행.
#####  -> 문제 분량을 다음주에 안하는 만큼 3문제 -> 6문제로 진행.
#####  -> 저번 주차 때 다같이 못푼 9267번 A + B 문제는 다음 주 까지 연장.
#####  -> 저번 주차 때 다같이 못푼 18111번 마인크래프트 문제는 다음 주 까지 연장.
#####  -> 모각코 인원중 창의작품 경진대회에 참여하는 인원이 있어 이번주는 미흡하였다.
#####  -> 이번주가 미흡한 만큼 다음주에는 해결못한 문제를 풀이하며 진행.
#
 
#### - 다음 시간까지 풀어올 문제를 선정

#####  -> 9267번 A + B 문제 (연장)
#####  -> 10773번 제로 문제
#####  -> 1016번 제곱 ㄴㄴ 수 문제
#####  -> 4179번 불! 문제
#####  -> 1267번 핸드폰 요금 문제
